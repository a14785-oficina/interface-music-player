## Enunciado

Reproduzir, em Figma, três ecrãs de uma aplicação mobile de music player, mantendo a máxima proximidade visual com as imagens de referência. Conteúdos exigidos pelo enunciado:

- Criar os ecrãs em formato mobile
- Reproduzir cores, tipografia, formas, ícones, gráficos, ilustrações, alinhamentos e espaçamentos
- Organizar corretamente layers e componentes
- Criar protótipo navegável básico

---

## Ecrãs Reproduzidos

| Ecrã | Características principais |
|---|---|
| Frame 1 | Biblioteca ou álbum em destaque — tipografia expressiva — paleta escura |
| Frame 2 | Player em reprodução — controlos de media (play, skip, shuffle) — barra de progresso — artwork do álbum |
| Frame 3 | Lista de reprodução — cards de músicas — metadados (título, artista, duração) |

---

## Processo de Trabalho

```
1. Análise das imagens de referência
   — Identificação das cores (paleta escura / dark UI)
   — Levantamento tipográfico (família, tamanho, peso)
   — Mapeamento de espaçamentos, alinhamentos e hierarquia

2. Construção em Figma
   — Frame mobile 390 × 844 px
   — Reprodução dos 3 ecrãs com fidelidade visual
   — Organização de layers: Background / Content / Controls / Navigation
   — Componentes reutilizáveis: botões, cards, ícones de media

3. Assets adicionais
   — Ícone de notificação (icons8 — 48px e 96px)
   — Integração de elementos gráficos complementares

4. Prototipagem
   — Ligação entre ecrãs com interações básicas

5. Exportação
   — PNG e SVG por ecrã (Frame 1, Frame 2, Frame 3)
   — Vista conjunta (Mobile.png / Mobile.svg)
```
